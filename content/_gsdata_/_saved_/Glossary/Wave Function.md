 Wave Function & Collapse
Core Concept
A wave function represents all possible states a quantum system can exist in-until measured, when it "collapses" into one definite outcome.
 Implication: Reality exists in multiple possibilities until observation forces a single outcome.

Key Elements
Superposition: Before collapse, an electron exists in all states at once ️.
Collapse by Observation: The moment a measurement is taken, all possibilities reduce to one reality.
